# Kago Hill Farm

## Getting Started

### Prerequisites

- Node.js 18+
- Yarn 1.x
- Kago Hill Farm API credentials

### Installation

1. Clone the repository:

2. Install dependencies:

   ```bash
   yarn install
   ```

3. Create `.env` file:

   ```bash
   cp .env.example .env
   ```

4. Start development server:

   ```bash
   yarn dev
   ```
